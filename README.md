# veg-quality-detector
##  Description

Veg Quality Detector is a machine learning-based web application that analyzes and classifies the quality of vegetables.
It combines a frontend interface with a Python backend and a trained ML model to provide predictions.

---

##  Features

*  Machine Learning-based quality detection
*  Uses trained model (`model.h5`)
*  Simple and interactive frontend
*  Backend processing using Python
*  Structured project with separate frontend & backend

---

## Technologies Used

* Python
* TensorFlow / Keras
* HTML
* CSS
* JavaScript
